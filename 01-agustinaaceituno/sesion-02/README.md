## Arduino 
Plataforma de software y de placas electrónicas 

## Arduino IDE
Software donde se escribe el código que correrá en nuestra placa.

Vamos a descargar la versión 2.3.8.
* Board - Arduino Uno R4 Boards
* Library - ArduinoMQTTCLIENT - Arduino Graphics

## Arduino Uno R4 Wifi
Placa integrada con wifi, conector USB-C y una pantalla de 12 x 8 pixeles.

Se puede hacer:
* Mandar información
* Conectarse a wifi
* Enviar datos 
* Usar la pantalla 
* Comunicarse con otros dispositivos

## Homebrew
Instalación https://brew.sh/. copiar el codigo/link, pegarlo en la terminal y ejecutarlo

**Terminal?**

## Terminal
Una aplicación que te permite controlar el computador escribiendo comandos de texto en vez de usar ventanas y botones gráficos.

Es como una “consola” desde donde puedes:
* ejecutar programas
* instalar cosas
* mover archivos
* programar
* controlar Arduino/Python/etc.
* usar herramientas de desarrollo

## MQTT
Protocolo de comunicación  utilizado principalmente en proyectos de IoT y automatización. Su funcionamiento se basa en el envío y recepción de mensajes entre dispositivos mediante un sistema.
