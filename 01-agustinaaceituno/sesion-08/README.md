## Phyton

CircuitPython es un lenguaje de programación diseñado para simplificar la experimentación y el aprendizaje de la programación en placas de microcontroladores de bajo coste.

## circuit python 10.2.0
sirve para manejar led, leer sensores, controlar pantallas OLED

Utilizamos la versión circuit python 10.2.0

Para empezar al conectarla aparece en el computador como CIRCUITPY, donde solo hay que ingresar archivos o modificarlos dentro de esta unidad.

Se debe instalar CircuitPython en la Raspberry Pi Pico
* en la pagina debemos seleccionar la placa que tenemos 
* conectamos la placa al computador
* abrir la unidad de la placa desde finder
* arrastrar el archivo descargado a la carpeta de la placa
* se reinicia la placa
