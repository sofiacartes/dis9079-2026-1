## MQTT 
_Message Queuing Telemetry Transport_

Es un protocolo ligero de mensajería diseñado para la comunicación entre dispositivos IoT, como Arduino, Raspberry Pi y sensores conectados a internet. Funciona sobre TCP/IP, permitiendo una conexión estable y continua entre dispositivos dentro de una red.


## HiveMQ
Plataforma de mensajería basada en MQTT utilizada para proyectos IoT. Funciona como un broker que permite conectar dispositivos y sistemas en la nube de manera rápida, segura y confiable, gestionando el intercambio de datos en tiempo real entre múltiples dispositivos.

Utilizado para guardar datos, configuraciones y archivos dentro de un sistema o dispositivo.

## Claude Shannon
Fue un matemático e ingeniero estadounidense considerado el padre de la teoría de la información. Sus investigaciones permitieron desarrollar las bases de las comunicaciones digitales modernas, explicando cómo transmitir información de manera eficiente entre dispositivos. Sus ideas influyeron directamente en tecnologías actuales como internet, redes Wi-Fi, compresión de datos, criptografía y sistemas de comunicación digital.
