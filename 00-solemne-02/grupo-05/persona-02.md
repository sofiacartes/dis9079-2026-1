# Sensor
## Sensor de sonido tipo KY-038 / LM393

### **¿Qué es?**
+ dispositivo electrónico que es capaz de **detectar ondas sonoras** del entorno y convertirlas en señales eléctricas, estas pueden ser interpretadas por un circuito, un microcontrolador o una computadora.
+ Utilizado en proyectos de robótica, domótica, instalaciones interactivas, instrumentos musicales electrónicos, sistemas de seguridad y arte multimedia.
+ El sonido es capaz de “viajar por el aire” en forma de vibraciones, en donde el sensor capta estas vibraciones mediante un componente llamado micrófono

### **Componentes**
- Micrófono
  + Captura vibraciones del aire.
  + Convierte sonido en pequeñas variaciones eléctricas.
    
- Amplificador
  + Aumenta señal del micrófono para poder procesarla.
    
- Comparador/Procesador
  + Determina si el sonido supera cierto umbral.
  + También se puede ajustar la sensibilidad con un potenciómetro (algunos sensores).
    
- Salida
  + Analógica: Entrega valores variables según intensidad del sonido.
  + Digital: Solamente indica si el sonido superó un nivel determinado.
