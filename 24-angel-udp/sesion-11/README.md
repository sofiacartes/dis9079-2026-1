## Correcciones

hay un componente llamada: RELÉ

es un interruptor eléctrico operado magnéticamente. Permite controlar un circuito de alta potencia (corriente fuerte) utilizando una señal de control de baja potencia (corriente débil), garantizando el aislamiento eléctrico entre ambos.

una pagina llamada **wokwi** paara hacer simulaciones sin tener componentes.
